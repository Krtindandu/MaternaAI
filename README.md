# MaternaAI

AI-Assisted Maternal Health Screening and Risk Assessment Platform

## Overview

MaternaAI is a mobile-friendly web application designed to support ASHA workers, healthcare professionals, and hospitals in the early identification of high-risk maternal health conditions.

The system assists in screening:

- Pre-eclampsia in pregnant women
- Postpartum Depression (PPD) in postpartum mothers

## Key Features

- AI-based risk scoring
- Multilingual support
  - English
  - Hindi
  - Telugu
- Voice-assisted questionnaire
- Maternal image capture support
- Doctor Dashboard
- Patient Records
- Emergency Alert System
- Risk Prioritization

## Tech Stack

Frontend:
- React.js
- Vite
- JavaScript
- Tailwind CSS

Browser APIs:
- Web Speech API
- Camera API (getUserMedia)

Storage:
- Local Storage

## Workflow

1. ASHA worker registers patient
2. Screening questionnaire conducted
3. AI calculates risk score
4. High-risk cases flagged
5. Doctor reviews patient
6. Doctor may:
   - Accept case
   - Transfer to another doctor
   - Refer to hospital
